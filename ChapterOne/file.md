#关于shell中文件的测试判断

[ -f $file ] 判断变量中是否包含一个合法的文件路径或文件名

[ -x $var ] 判断变量中是否包含可执行的文件

[ -d $var ] 判断变量中包含的是否是一个目录

[ -e $var ] 判断变量中包含的文件是否存在

[ -c $var ] 判断变量是否是一个字符设备文件

[ -b $var ] 判断变量是否是一个块设备文件

[ -w $var ] 判断变量是否是一个可写的文件

[ -r $var ] 判断变量是否是一个可读的文件

[ -L $var ] 判断变量是否是一个符号链接

#字符串比较

[[ $str1 = $str2 ]]
[[ $str1 == $str2 ]]
[[ $str1 != $str2 ]]
[[ $str1 > $str2 ]]
[[ $str1 \< $str2 ]]
[[ -z $str1 ]]  是空返回真
[[ -n $str1 ]]  是非空返回真
